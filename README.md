E-commerce Async

Projeto simples demonstrando o uso de XMLHttpRequest para requisições assíncronas a uma API REST.

## Funcionalidade
- Requisição assíncrona com XMLHttpRequest
- Consumo de API REST
- Atualização dinâmica da interface

## Como usar
1. Abrir o arquivo index.html no navegador
2. Clicar no botão "Carregar dados"

## Tecnologias
- HTML
- JavaScript
